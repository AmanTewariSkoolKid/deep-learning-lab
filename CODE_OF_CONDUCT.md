# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Pledge
We pledge to foster an open and welcoming environment.

## Our Standards
Examples of positive behavior include:
- Inclusive language
- Constructive feedback
- Focus on collaborative learning

Unacceptable behaviors include harassment, insults, or discrimination.

## Enforcement
Report concerns via GitHub issues or direct maintainer contact. Maintainers will act in good faith to resolve conflicts.
